POJ3417 Network

POJ3417

​ 给你一个节点为n的(n<1e5)树，再给你m(m<1e5)条在这棵树上新加的边，问你在只能删除一条原来树上的边和一条新边的情况下，最多有多少种方案可以使得图被划分成两个连通块。

题解：

​ 毫无疑问，新加入的边一定会使得图中出现环，如果树的旧边被一个以上的环覆盖，那么不可能满足分割图的要求，如果被一个环覆盖，那么可以删除一条新边使得不再成环，再删除一条该旧边使得图被分割，故方案数+1，如果一条旧边不被环覆盖，那么方案树+m(即删除任意一条新边)，具体实现：用一个数组cir_num[i]表示编号为i的点的父边被多少个环覆盖，再根据差分的思想，每加入一条新边edge(u,v)，将两点环覆盖数+1，且lca环覆盖数-2，保证整差分的正确性，cir_num[u]++,cir_num[v]++,cir_num[lca(u,v)]-=2,最后一次dfs，将子树的cir_num[v]累加至父亲节点u上，然后判断环覆盖数，特判一下根节点(cir_num数组的含义)，累加答案即可。

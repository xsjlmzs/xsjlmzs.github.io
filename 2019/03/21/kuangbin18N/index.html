kuangbin后缀数组专题18 N

N

​ 给你一串数字，保证这串数字的第一个数字是最大的且没有之一。你的任务是把这个字符串分成三段且每段不能为空。然后将这三段反转，使反转后的这串数字字典序最小。长度<=2e5。

题解：后缀数组 / 最小表示法

​ 这题没有给数字的大小范围，所以我们需要度进来之后先离散化一下以免RE。我们先将原串反转，反转的目的是可以利用后缀数组找到第一段反转的位置one_pos，只需要找出反转之后的字典序最小后缀即可，这里注意还有一个细节是这个后缀的起始位置必须>1(从0开始的存的话)，因为这样才能使接下来有第二次第三次翻转，否则二三次翻转为空不满足题意。这样第一步已经完成，我们的任务变成了去找第二个最佳反转位置，这次还能像第一次那样找字典序最小的就是答案吗？
